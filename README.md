# DVRTS - A VR Darts Game

<p align="center">
  <img src="img/dvrts_logo.png" alt="DVRTS" width="400"/>
</p>

A virtual reality darts game developed as part of the DH2310 Extended Reality in Theory and Practice course at KTH. Players can pick up and throw darts at a board in an immersive VR environment. This project demonstrates XR interaction design, 3D modeling, and VR gameplay mechanics.
